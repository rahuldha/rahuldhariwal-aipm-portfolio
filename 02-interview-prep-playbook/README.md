## How to Use This

**Before a phone screen**: Read the relevant company prep file + the behavioral question bank. Practice the 60-second "tell me about yourself" answer out loud.

**Before a technical interview**: Work through the Technical AI question bank. Know the RAG, fine-tuning, and evaluation framework answers cold.

**Before a product design round**: Review the Product Strategy question bank + the relevant STAR stories. Map each story to 2-3 different question types — the same story should be usable for "failure," "influence without authority," and "metrics" questions.
